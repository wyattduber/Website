My Self-Hosted Sites & Projects

This is a collection of all of the various websites and projects that I am self-hosting. These are not included in my Projects Page, as those are programming projects and these are just apps. Everything you see on this list is hosted on my own hardware, from my apartment. Click a logo to see the live site, and the info box to find the origin of the application! Click me to view a list of all of my hosted sites!

Nextcloud

This is a self-hosted file storage application that I use to store and back up files that I deem important to keep around. It is hosted on my own hardware in my apartment.

Plex

This is a self-hosted media storage application that I use to store movies, TV, music, and other important videos/media.

Upptime

Upptime is a status page based on GitHub Pages and Actions. It tracks all of my other subdomains and sites, checking their uptime.

Paste

Paste is a quick and light weight pastebin-like tool that can be used to share code or text files. I self-host this one so that I don't have to risk any private data being pasted into a site that I do not own.

Docs

This is a simple website where I am hosting all of my documentation (generated or not) for different projects. Things like Javadoc.

AMP (Application Management Panel)

AMP is a game server hosting platform application that makes dedicated game server hosting easy, simple, and containerized. I self host it because it is convenient for hosting servers for others.
